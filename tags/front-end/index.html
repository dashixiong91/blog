Flutter Versions Manager

@大师兄 · Oct 31, 2019 · 1 min read

fvm Flutter SDK 多版本管理工具 安装 首先，添加homebrew的tap源： brew tap xinfeng-tech/fvm 添加完tap之后，你可以安装fvm brew install fvm 拷贝以下内容到你的 .bashrc|.zshrc ... 文件中 export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn export FVM_DIR="$HOME/.fvm" source "/usr/local/opt/fvm/init.sh" 用例 打印 fvm 帮助信息 fvm # or `fvm --help` 安装 Flutter某个版本，以1.9.1为例 fvm install 1.9.1 使用上面安装的版本 fvm use 1.9.1 设置别名 fvm alias latest 1.9.1 列出已安装的所有版本 fvm list 列出已发布的版本 fvm list-remote all 预览图 链接 Github Repo
